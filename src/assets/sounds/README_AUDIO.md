# Arquivos de Áudio Necessários

Para que o jogo tenha som, você precisa adicionar os seguintes arquivos de áudio nesta pasta (`src/assets/sounds/`):

1. **ambience.mp3** - Som de torcida/estádio (loop)
2. **ball_kick.mp3** - Som do chute na bola
3. **goal.mp3** - Som de grito de gol e comemoração
4. **miss.mp3** - Som de decepção/defesa do goleiro
5. **whistle.mp3** - Som de apito do juiz
6. **selection.mp3** - Som de clique/seleção de opção
7. **ball_post.mp3** - Som da bola batendo na trave
8. **victory.mp3** - (Já existente) Som de vitória
9. **defeat.mp3** - (Já existente) Som de derrota

Certifique-se de que os nomes dos arquivos estejam exatamente em letras minúsculas como listado acima.
